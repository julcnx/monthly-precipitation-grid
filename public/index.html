Monthly Precipitation Grid

Coarse 2.5° monthly precipitation climatology (NOAA CMAP). Click the map to set a route start (A) then end (B); routes are requested from a public Valhalla instance (see README).

January
Click map to pick start point.
Hover a cell to see its monthly value.
API
GET /api/precip?lat=&lon=&month=
Value for one point.
curl 'localhost:3000/api/precip?lat=19.9&lon=99.8&month=8'
GET /api/cells?month=&bbox=
GeoJSON grid, whole globe or one bbox (minLon,minLat,maxLon,maxLat).
curl 'localhost:3000/api/cells?month=8&bbox=98,13,101,20'
GET /api/route?lat1=&lon1=&lat2=&lon2=
Proxies to Valhalla, powers this page's route demo only.

Full integration guidance (bake into the graph at build time vs. bbox fallback) is in the README.
